--- [NVIDIA Vanta LT Specs | TechPowerUp GPU Database] ---
The Vanta LT was a graphics card by NVIDIA, launched on March 1st, 2000. Built on the 250 nm process, and based on the NV5 graphics processor, in its Vanta LT variant, the card supports DirectX 6.0. Since Vanta LT does not support DirectX 11 or DirectX 12, it might not be able to run all the latest games. The NV5 graphics processor is a relatively small chip with a die area of only 63 mm² and 15 million transistors. It features 2 pixel shaders and 0 vertex shaders, 2 texture mapping units, and 2 ROPs. Due to the lack of unified shaders you will not be able to run recent games at all (which require unified shader/DX10+ support). NVIDIA has paired 8 MB SDR memory with the Vanta LT, which are connected using a 64-bit memory interface. The GPU is operating at a frequency of 105 MHz, memory is running at 100 MHz. Being a single-slot card, the NVIDIA Vanta LT does not require any additional power connector, its power draw is not exactly known. Display outputs include: 1x VGA. Vanta LT is connected to the rest of the system using an AGP 2x interface.
