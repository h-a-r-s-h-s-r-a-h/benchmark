--- [NVIDIA GeForce4 MX 420 Specs | TechPowerUp GPU Database] ---
The GeForce4 MX 420 was a graphics card by NVIDIA, launched on February 6th, 2002. Built on the 150 nm process, and based on the NV17 graphics processor, in its MX420 variant, the card supports DirectX 7.0. Since GeForce4 MX 420 does not support DirectX 11 or DirectX 12, it might not be able to run all the latest games. The NV17 graphics processor is a relatively small chip with a die area of only 65 mm² and 29 million transistors. It features 2 pixel shaders and 0 vertex shaders, 4 texture mapping units, and 2 ROPs. Due to the lack of unified shaders you will not be able to run recent games at all (which require unified shader/DX10+ support). NVIDIA has paired 64 MB SDR memory with the GeForce4 MX 420, which are connected using a 128-bit memory interface. The GPU is operating at a frequency of 250 MHz, memory is running at 166 MHz. Being a single-slot card, the NVIDIA GeForce4 MX 420 does not require any additional power connector, its power draw is not exactly known. Display outputs include: 1x VGA, 1x S-Video. GeForce4 MX 420 is connected to the rest of the system using an AGP 4x interface.

NVIDIA GeForce4 MX 420 GPU NVIDIA GeForce4 MX 420 desktop Graphics Processing Unit was released in February 2002. The GPU is built upon NVIDIA Video Processing Engine architecture, and it is produced on 150 nm technology. The card's graphics frequency is 250 MHz. Furthermore, it has 2 pixel shaders, 4 texture units, and 2 ROPs. The GeForce4 MX 420 embeds 64 MB of DDR memory. Because the memory operates at 166 MHz, and uses 64 bit bus, the effective memory bandwidth is 2.66 GB/s. The GPU has AGP 4x interface, and needs a single slot on the motherboard.
