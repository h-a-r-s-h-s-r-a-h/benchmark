[Robe - Official Terraria Wiki]
The Robe is a vanity item equipped in the shirt slot. It resembles the robes worn by Dark Casters and Tim. It can be augmented by gems to form part of the Wizard set.
URL: https://terraria.wiki.gg/wiki/Robe

[Wizard set - Official Terraria Wiki]
The Mystic Robe and Wizard Hat combination is considered the strongest pre-Hardmode armor for magic users not interested in defense, as their offensive stats greatly surpass those of both Jungle and Meteor armor.
URL: https://terraria.wiki.gg/wiki/Wizard_set

[Magic Hat - Terraria Wiki]
The Magic Hat is an armor item that can be equipped in the helmet slot. It increases the player's magic damage and critical strike chance by 6% / 7% each, and grants 0 additional defense.
URL: https://terraria.fandom.com/wiki/Magic_Hat

Wizard set Jump to navigation Jump to search Wizard set Statistics | Type | | |---|---| | Set Bonus | : Increases maximum mana by 60 : 10% increased magic critical strike chance | The Wizard set is an armor set consisting of one of the magic headgear and one of the magic robes. The robe can be any of the seven gem robes, crafted using a Robe and gems, or the Mystic Robe, bought from the Traveling Merchant. Magic headgear can be either the Wizard Hat, which is dropped by Tim, or the Magic Hat, which is occasionally sold by the Traveling Merchant.
